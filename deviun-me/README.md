# Web

## NEEDS

- API SERVER

## ENV

Make file `.env` at root folder.

`API_SERVER` — URL to API server for frontend.
`INTERNAL_API_SERVER` — Relative URL to API server for backend.

## INSTALL

```bash
$ npm i
```

## PROD BUILD

```bash
$ npm run build
```

## RUN

Developer mode:

```bash
$ npm run dev
```

Production mode:

```bash
$ npm start
```
